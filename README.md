# Repository with Code and Data from R Workshop
This repository has the data and code from an R Workshop run by Muji Chughtai (PhD Candidate in Psychology) for interns/RAs in the Yale Psychology Department. The code gives a brief introduction to using R for data cleaning, data visualization, and data analysis. The data is from the World Bank from 2016, and it includes general and economic indicators for countries around the world.

To learn more about this dataset, see https://data.worldbank.org/indicator
